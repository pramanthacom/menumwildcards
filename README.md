# MENUM Wild Cards

Онлайн-витрина метафорических карт MENUM для фестиваля Wild Ecstatic Dance.

Публичная версия: https://menum.space/menumwildcards/ (зеркало: https://pramanthacom.github.io/menumwildcards/)

Короткие ссылки карт (зашиты в QR-коды): `https://menum.space/card/<номер-слаг>`, например https://menum.space/card/01-vstrecha — nginx menum.space редиректит их на `web/preview-<номер-слаг>.html`.

В репозитории хранятся только файлы, необходимые для просмотра сайта. Рабочие документы, таблицы и служебные материалы исключены из публикации.
